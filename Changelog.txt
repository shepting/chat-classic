# ChatGPT Classic - Changelog

## Version 5.0 (2025-08-24)
### Modular Bazel Architecture
- **Separate Bazel Modules**: Each component now has its own BUILD file in Sources/ directory
- **Cross-Module Dependencies**: Proper dependency resolution between App, MainWindow, and component modules
- **Header Visibility**: Fixed include paths with `includes = ["."]` for cross-module imports
- **Circular Dependency Resolution**: App module uses runtime class loading to avoid import cycles
- **Individual Module Builds**: All modules (App, HistoryTable, ChatConversation, ChatInput, Networking) build independently
- **Full Application Build**: Complete application builds successfully with modular architecture
- **Runtime Compatibility**: Application launches and runs with separated module structure
- **Build System Optimization**: Streamlined Bazel configuration for modular development

## Version 4.0 (2025-08-24)
### Modular Architecture Refactor
- **App Module**: Centralized application management with singleton pattern
- **HistoryTable Module**: Dedicated conversation sidebar with delegate pattern
- **ChatConversation Module**: Structured message display with ChatMessage model
- **ChatInput Module**: Reusable input component with placeholder behavior
- **Networking Module**: Simulated API communication with delegate callbacks
- Refactor MainWindowController to use composition over inheritance
- Implement delegate patterns for inter-module communication
- Add conversation switching functionality in sidebar
- Add simulated chatbot responses based on message content
- Improve code organization and separation of concerns
- Enhanced memory management with proper module lifecycle

## Version 3.0 (2025-08-24)
### Enhanced Input and Styling
- Replace NSTextField with NSTextView for multi-line chat input
- Increase input area height from 60px to 80px for better usability
- Add Aqua-style Send button with NSRoundedBezelStyle
- Implement placeholder text behavior for NSTextView ("Ask anything")
- Add Enter key equivalent for Send button
- Include proper focus handling with textDidBeginEditing/textDidEndEditing
- Add scroll view wrapper for text input with auto-hiding scrollers

## Version 2.0 (2025-08-24)

## Version 1.0 (2025-08-24)
### Initial Release - Legacy macOS AppKit Project
- AppKit application targeting macOS 10.4 Tiger
- Split view interface with sidebar (NSOutlineView) and chat table (NSTableView)
- Bazel build system using rules_apple 4.1.2
- Manual memory management (retain/release, no ARC)
- Comprehensive documentation for macOS 10.2-10.4 AppKit features
- Basic window with sidebar showing conversation list
- Chat messages display area
- Legacy compatibility with PowerPC and Intel Macs
- Complete Bazel build configuration with MODULE.bazel and WORKSPACE support

## Technical Notes
- **Target Platform**: macOS 10.4 Tiger and later
- **Architecture**: PowerPC and Intel (legacy compatibility)
- **Memory Management**: Manual retain/release (no ARC)
- **Build System**: Bazel with rules_apple 4.1.2
- **UI Framework**: AppKit with programmatic layout
- **Language**: Objective-C
